# Приложение для оценки ответов тестируемого на основе модели ML
## Для разработки данного приложения использован фреймворк FastApi имеющий встроенную документацию и ускорению работы благодаря встроенной асинхронности, что важно при работе с ML моделями
## Для запуска проекта локально:
* Скопировать репозиторий командой git clone https://github.com/ALook366/Sigma_case_ex.git
*  Создать и активировать виртуальное окружение (рекомендуется)  
  python -m venv venv  
  Активировать venv:  
  В Windows:  
  .\venv\Scripts\activate  
  В Linux/macOS:  
  source venv/bin/activate  
  Перейти в директорию Sigma_case_ex командой  cd .\Sigma_case_ex\
  Установить зависимости  
  pip install -r requirements.txt(или см. в проекте аналогичное название файла)
  ### Примечание:  
  При установке зависимостей может возникнуть переполнение и нагрузка на системный диск в папке temp, в таком случае можно использовать дополнительный выриант:
  Выполнить в консоле IDE:
  New-Item -ItemType Directory -Force -Path "D:\temp"  
New-Item -ItemType Directory -Force -Path "D:\pip_cache"   
New-Item -ItemType Directory -Force -Path "D:\pip_build"  

$env:TMP = "D:\temp"  
$env:TEMP = "D:\temp"   
$env:PIP_BUILD_DIR = "D:\pip_build"  

pip install --cache-dir D:\pip_cache -r requirements.txt
*  перейти в корневую директорию клонированного репозитория и выполнить команду:  
   python -m uvicorn Sigma_case.app.main:app --reload
   Перейти в адресной строке браузера по адресу localhost:8000
## После загрузки файла можно  выбрать количество строк для обработки из исходного файла, в результате пользователю будет доступно для скачивания два файла:
  * Полный файл с текстом и транскрибцией вопросов, выставленными оценками
  * Краткий файл с итогами  на основе полного файла с id экзамена, id вопроса и оценки
### Документация и API доступна по адресу localhost:8000\docs при запущеном приложении
